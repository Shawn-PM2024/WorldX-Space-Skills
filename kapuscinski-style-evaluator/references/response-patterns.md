# Response Patterns

Use this file when you need compact working shapes for responses.

## 1. Default Shape

### Overall editorial judgment

Open with:

- what kind of draft this is
- in what sense it moves closer to or stays distant from the target manner
- what the central craft problem is

### Detailed commentary

Choose only a few issues. Good categories include:

- scene entry
- detail weight
- narrative distance
- pacing
- premature abstraction
- over-explanation
- forced elevation

Do not rigidly surface the internal framework unless the user asks for it.

Prefer:

- a natural editorial flow
- one central judgment, then two or three supporting craft observations
- prose that sounds like a letter from an attentive editor

Avoid:

- "Observation layer:"
- "Narration layer:"
- "Meaning-generation layer:"
- obvious rubric headings unless explicitly requested

### Practice suggestions

End with 1-3 exercises that are small, local, and immediately actionable.

### Optional micro-example

When helpful, end with a tiny illustrative passage:

- one to three sentences only
- clearly framed as an illustration
- used to help the reader feel the craft difference
- never expanded into a full imitation paragraph

## 2. Tone Switching

### Default mode

Use a calm, editorial, helping tone.

Useful phrasing:

- "This passage moves closer to..."
- "What is working here is..."
- "What still keeps the paragraph at a distance is..."

### Stricter mode

If the user explicitly asks for stricter feedback, become clearer and firmer without becoming theatrical or punitive.

Useful phrasing:

- "At the moment this reads more like explanation than observation."
- "The paragraph wants significance, but it has not yet earned it."
- "The material is interesting, but the prose is still telling rather than discovering."

## 3. Out-Of-Range Handling

When the input is only partly suitable:

1. name the mismatch briefly
2. preserve what can still be evaluated
3. avoid totalizing judgment

Useful phrasing:

- "This is not an ideal input for the full evaluator, but one comparable layer is..."
- "As a poem, it falls outside the main range of this skill, though its image logic can still be discussed."
- "As a thesis-first op-ed, it is less useful to ask whether it resembles Kapuscinski broadly; the more productive question is whether it allows observation any real work."

## 4. Local Rewrite Limits

By default:

- do not rewrite

If explicitly requested, or if a tiny example would substantially clarify the conclusion:

- use one sentence or a tiny passage
- show the craft move
- do not output full-paragraph mimicry

Useful framing:

- "Here is a tiny local example of the move, not a replacement passage."
- "This is meant to demonstrate pacing and observational weight, not to imitate the author's voice wholesale."
- "This miniature example is only here to make the difference more tangible on the page."
